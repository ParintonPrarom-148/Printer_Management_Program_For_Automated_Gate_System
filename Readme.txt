

# 🖨️ Printer Management Program for Automated Gate System

A C# Windows Forms application designed to manage printer-related tasks for an automated gate control system. This program ensures that printing activities (e.g., tickets, logs, or gate passes) are recorded and monitored effectively.

## 🔧 Features

- Monitor and log printer activity
- Export print logs (e.g., JSON or TXT)
- Integrate with CUPS (via WSL or Linux backends)
- Minimal user interface for practical deployment
- Useful for smart gate or parking automation systems

## 🧰 Technologies

- C# (.NET Framework)
- Windows Forms
- (Optional) CUPS for Linux/WSL integration
- JSON File Logging
